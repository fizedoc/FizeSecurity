check()
-------
验证数据

.. code-block:: php

  public function check (
      array $data
  ) : bool|string|array


:参数:
  +-------+----------------+
  |名称   |说明            |
  +=======+================+
  |data   |待验证数据      |
  +-------+----------------+
  
  

:返回值:
  成功返回 true，失败返回失败信息


::

    如果启用批量验证，则失败时返回值为错误数组，否则为错误信息

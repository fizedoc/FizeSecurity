inIp()
------

.. code-block:: php

  public static function inIp (
      string $ip,
      array $networks
  ) : bool


:参数:
  +---------+-------------+
  |名称     |说明         |
  +=========+=============+
  |ip       |             |
  +---------+-------------+
  |networks |网段数组     |
  +---------+-------------+
  
  


::

    网段参数支持如下格式：
    - 单个 IP : 192.168.5.1
    - 带 * 号通配符 IP 格式：192.*.*.1
    - IP 段组：192.168.5.1 - 192.168.10.101

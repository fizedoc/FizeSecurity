hash()
------
创建密码的散列

.. code-block:: php

  public static function hash (
      string $password,
      int $algo,
      array $options = null
  ) : string


:参数:
  +---------+----------------------------------+
  |名称     |说明                              |
  +=========+==================================+
  |password |密码                              |
  +---------+----------------------------------+
  |algo     |指示算法的密码算法常量            |
  +---------+----------------------------------+
  |options  |选项                              |
  +---------+----------------------------------+
  
  

:返回值:
  失败时返回 false
